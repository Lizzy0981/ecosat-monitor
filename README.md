## ✨ Características Principales

### 🛰️ **Datos Satelitales en Tiempo Real**
- **Landsat-8**: Imágenes multiespectrales, NDVI, análisis de vegetación (30m resolución)
- **Sentinel-2**: Monitoreo de deforestación y cambios de uso de suelo (10m resolución)
- **MODIS Terra/Aqua**: Temperatura superficial terrestre (LST) y detección de focos de calor
- **Sentinel-5P**: Calidad del aire (NO2, CO, CH4, SO2) y gases atmosféricos
- **Próximos Pases**: Horarios precisos de satélites sobre zonas específicas

### 🌍 **Zonas Ambientales Monitoreadas**
- **Reserva Amazónica Central** (Brasil): NDVI 0.89 - Estado excelente
- **Bosque Boreal Canadiense**: NDVI 0.85 - Muy saludable  
- **Taiga Siberiana** (Rusia): NDVI 0.78 - Monitoreo de incendios
- **Selva del Congo** (RDC): NDVI 0.82 - Análisis de deforestación

### 📊 **Métricas Ambientales Avanzadas**
- **NDVI (Normalized Difference Vegetation Index)**: 0.0-1.0 escala satelital
- **LST (Land Surface Temperature)**: Temperatura superficial desde satélites
- **Focos de Calor**: Detección de incendios en tiempo real via MODIS
- **Gases Atmosféricos**: NO2, CO, CH4, SO2 desde Sentinel-5P
- **Cobertura Nubosa**: Porcentaje de nubes que afecta observaciones

### 🎮 **Sistema de Gamificación**
- **Logros Satelitales**: Maestro Satelital, Rastreador NDVI, Observador de Datos
- **Sistema de Puntos**: +350 pts por análisis satelital completo
- **Niveles de Rareza**: Legendario, Épico, Raro con diferentes recompensas
- **Seguimiento de Rachas**: Días consecutivos monitoreando sostenibilidad
- **Desafíos**: Metas específicas por zona ambiental

### 🌍 **Sistema Bilingüe Completo**
- **Español/English**: Switching dinámico sin recargar página
- **Términos Satelitales**: Traducción técnica de métricas NDVI, LST, etc.
- **Localización**: Fechas, números y formatos según región
- **Nombres de Satélites**: Traducción de Landsat, Sentinel, MODIS
- **Dynamic Loading**: Carga bajo demanda de traducciones

### 📱 **PWA y Funcionalidad Offline**
- **Service Worker**: Cache inteligente de datos satelitales
- **IndexedDB Storage**: 7 días de datos satelitales offline
- **Background Sync**: Sincronización automática de pases satelitales
- **Modo Offline**: Funcionalidad completa sin conexión
- **Installable**: Instalación como app nativa en dispositivos

## ⚡ Instalación y Uso

### **Configuración Local**
```bash
# 1. Clonar repositorio
git clone https://github.com/Lizzy0981/ecosat-monitor.git
cd ecosat-monitor

# 2. Servidor local (opcional)
# Opción A: Python
python -m http.server 8000

# Opción B: Node.js
npx http-server -p 8000

# Opción C: Live Server (VS Code)
# Opción D: Abrir directamente index.html

# 3. Acceder
# http://localhost:8000
```

### **Configuración de APIs Satelitales**
```javascript
// En src/data-manager.js - API keys opcionales
const satelliteConfig = {
  nasa: {
    apiKey: 'DEMO_KEY', // Gratis: 1000 req/hora

  }
};
```

```yaml
Satellite Data Processing:
  - NDVI Calculation: (NIR - RED) / (NIR + RED)
  - LST Processing: Thermal band calibration
  - Change Detection: Multi-temporal analysis
  - Cloud Masking: Automated cloud filtering
  - Geometric Correction: Orbital positioning

APIs Integration:
  - NASA EarthData: Landsat-8, MODIS Terra/Aqua
  - ESA Copernicus: Sentinel-2, Sentinel-5P
  - NOAA: Climate data, vegetation health
  - USGS: Landsat archive, change detection
```
